Javascript Window Manager

Full featured JavaScript Window Manager with a UNIX-like desktop environment style.
Uses PHP and MySQL as a backend to serve applications and resources. Contains an application API, and uses AJAX is used for event handling and system calls.

Features

  • CSS3 styling
    HTML5: document, localStorage, audio, video, canvas
  • Desktop
  • Desktop Panel with alignment options
  • Panel items:
    • Menu
    • Separator
    • Window List
    • Launcher dock
    • Clock
  • Windows with Applications
  • Dialogs
  • Operation dialogs:
    • Color picker
    • File copy
    • File upload
    • File open / save
    • File rename
  • Context menus, window menus, window statusbar
  • Session managment (saving and restoring of desktop window instances)

Applications

  • Clock
  • Draw
  • File Manager
  • Textpad
  • Terminal
  • Viewer (audio/video/images
  • System: Settings, About, User info, Logout
